We recommend to go over the examples in the following order:

1) example_basic.py - example discussed in the documentation.

2) example_all_scenes.py - rendering of all currently available scenes.

3) example_minimal.py - (almost) minimal configuration file.

4) example_param_types.py - passing of different types of values of parameters.

5) example_putting_it_all_together.py - all the main features, including advanced use of Python code.
